Note:How to create VirtualDirectory

1.Open Internet Information Services (IIS) Manager

2.Right click on Default Website

3.Select Add Virtual Directory option

4.Give alias name as any name for your directory(ex-priscilla)

5.browse path to save it in a folder

6.now write your program in notepad or any other editor and save it under virtual directory folder

7.open broswer and give (http://localhost/virtual directory name/filename)ex:htt://localhost/priscilla/vsdirex.html

8.We can see the output
